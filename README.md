# TheDCo Toolkit

## Overview

TheDCo Toolkit is an open-source business operating system designed to help creative entrepreneurs build and manage DIY kits, handmade products, supplier networks, product development, inventory, pricing, content creation, and business operations.

This project is being built step by step as a real-world system for TheDCo.
